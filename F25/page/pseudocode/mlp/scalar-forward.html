**Forward Pass** ~ Input: $D$ dimensional vector $\mathbf{x} = [x_j, j=1..D]$ $\\$ Set: $\\$ $\quad $ $D_0=0,$ is the width of the $0^\text{th}$ (input) layer $\\$ $\quad $ $y_j^{(0)}=x_j,$ $j=1..D$ $\\$ $\quad $ $y_0^{(k=1..N)}=x_0=1$ $\\$ For layer $k = 1..N$ $\\$ $\quad $ For $j=1..D_k$ $\\$ $\qquad$ $z_j^{(k)}=\sum_{i=0}^{N_k}w_{i,j}^{(k)}y_i^{(k-1)}$ $\\$ $\qquad$ $y_j^{(k)}=f_k(z_j^{(k)})$ $\\$ Output: $\\$ $\quad $ $Y=y_j^{(N)},$ $j=1..D_N$ $\\$
